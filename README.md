## 📖 About
This repository contains a comprehensive collection of Data Structures and Algorithms (DSA) solutions implemented in Java. The problems are sourced from popular coding platforms like LeetCode and GeeksforGeeks, covering various topics and difficulty levels.

## 🚀 Problems Solved

### Arrays & Two Pointers
- **Two Sum** - Find two numbers that add up to a target
- **Contains Duplicate** - Check if array has duplicate elements
- **Move Zeros** - Move all zeros to the end of array
- **Remove Duplicates** - Remove duplicates from sorted array
- **Merge Two Sorted Arrays** - Merge two sorted arrays efficiently
- **Sort Colors** - Dutch National Flag problem
- **Container With Most Water** - Two pointer optimization problem

### Binary Search
- **Binary Search** - Classic binary search implementation
- **Search Insert Position** - Find insert position in sorted array
- **Peak Element** - Find peak element in array
- **Search in Rotated Sorted Array** - Binary search variant
- **Search in 2D Matrix** - 2D binary search
- **Minimum in Rotated Sorted Array** - Find minimum efficiently

### Sliding Window
- **Maximum Subarray (Kadane's Algorithm)** - Find maximum sum subarray
- **Best Time to Buy and Sell Stock** - Stock profit optimization
- **Longest Substring Without Repeating Characters** - Sliding window technique
- **Maximum Points from Cards** - Card selection optimization
- **Sliding Window Maximum** - Find maximum in sliding windows
- **Nice Subarrays** - Count subarrays with k odd numbers
- **Max Consecutive Ones** - Find longest sequence of 1s

### Hashing & Prefix Sum
- **Valid Anagram** - Check if two strings are anagrams
- **Group Anagrams** - Group strings that are anagrams
- **Longest Consecutive Sequence** - Find longest sequence
- **Subarray Sum Equals K** - Count subarrays with given sum
- **Continuous Subarray Sum** - Check for continuous sum divisible by k
- **Running Sum of 1D Array** - Calculate prefix sum

### Miscellaneous
- **Reverse String** - String reversal
- **Missing Number** - Find missing number in sequence
- **Second Highest** - Find second largest element
- **Kth Largest Element** - Find kth largest using heap
- **Stock Span Problem** - Calculate stock span (GFG)
